# Upendra Raj Joshi — Portfolio

A polished personal portfolio website built with React, TypeScript, Vite, and Tailwind CSS. This portfolio showcases backend-focused projects, technical experience, certificates, and contact links with smooth animations and responsive navigation.

Live link: https://upendra-portfolio-web.vercel.app/

## 🚀 Overview

- Modern portfolio site for Upendra Raj Joshi
- Responsive layout with desktop sidebar navigation and mobile menu
- Animated sections using `motion`
- Project showcase with image previews, links, and tech stack tags
- Contact CTA with email and LinkedIn actions

## 🧩 Technologies

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- `motion` for animations
- `lucide-react` for icons
- ESLint for linting

## 🧭 Key Sections

- **Hero**: landing introduction with call to action
- **About**: bio, backend skill summary, and tech stack
- **Experience**: professional journey and achievements
- **Showcase**: selected projects with previews, GitHub links, and live demos
- **Certificates**: certification highlights
- **Contact**: CTA and direct email / LinkedIn links

## 💻 Local Development

```bash
npm install
npm run dev
```

Then open the local Vite server URL shown in the terminal.

## 📦 Build

```bash
npm run build
npm run preview
```

## 📁 Project Structure

- `src/` — application source files
- `src/components/` — reusable UI components
- `src/sections/` — page sections like `Hero`, `About`, `Showcase`, and `Contact`
- `src/assets/images/` — image assets used in the site
- `src/styles/index.css` — global styling
- `vite.config.ts` — Vite configuration
